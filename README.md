### Software Developer
Software developers use a variety of different coding languages to create their own software. Everyday they are challenged with a different task. However, some of the main tasks they face include overseeing software programming, testing software, laying out software and mapping out future software updates. Many software developers have a degree in computer science, this helps lay down the basics of coding. 
### Technical Skill
* Have a good understanding of Linux
* Have a good understanding of different programming languages
    * Python
    * Java
    * C#
    * SQL
    * HTML
* Software engineering skills
* Development operations skills

### Non-Technical Skills:
* Problem solving skills
* Communication skills
* Analytical thinking

## Standard Education
### University of Waterloo (Computer Science)
<img src="https://cs.uwaterloo.ca/sites/ca.computer-science/files/uploads/images/cheriton-school-computer-science.jpg" width="650" height="400">

### Entrance Requirements
* 6 Grade 12 U and/or M courses including…
    * Advanced Functions
    * Calculus and Vectors
    * Any Grade 12 U English
    * One other 4U course (Recommended: Grade 11 U Introduction to Computer Science)

### First Year Programs 
* [CS 135](https://ucalendar.uwaterloo.ca/2223/COURSE/course-CS.html#CS135) (Designing Functional Programs): Introduction to the basics of coding. Examples: Syntaxes, tracing, design, testing and data structures, etc…
* [MATH 135](https://ucalendar.uwaterloo.ca/2223/COURSE/course-MATH.html#MATH135) (Algebra for Honors Mathematics): Introduction to the language and proof techniques using algebra, integers, integer modulo, the rational number, etc…
* [MATH 137](https://ucalendar.uwaterloo.ca/2223/COURSE/course-MATH.html#MATH137) (Calculus 1 for Honors Mathematics): Introduction to series, limits, intermediate value theorem and approximate solutions to equations, derivatives, etc…
* [CS 136](https://ucalendar.uwaterloo.ca/2223/COURSE/course-CS.html#CS136) (Elementary Algorithm Design and Data Abstraction): Takes the basics learned from CS 135 and builds on it while transitioning to using interpretive language.
* [CS 136L](https://ucalendar.uwaterloo.ca/2223/COURSE/course-CS.html#CS136L) (Tools and Techniques for Software Development): Introduction to tools and techniques that help the software development life cycle.
* [MATH 136](https://ucalendar.uwaterloo.ca/2223/COURSE/course-MATH.html#MATH136) (Linear Algebra 1 for Honors Mathematics): Teaches concepts such as systems of linear equations, matrix algebra, elementary matrices, computational issues, etc… 
* [MATH 138](https://ucalendar.uwaterloo.ca/2223/COURSE/course-MATH.html#MATH138)  (Calculus 2 for Honors Mathematics): Introduction to concepts such as riemann integral and approximations, antiderivatives and the fundamental theorem of calculus, change of variables, methods of integration, etc…
* One communication skills course
* 3 electives

To obtain your degree, it takes roughly 5 years. If you choose to do the Co-op program, it is two years of paid work and experience.

## Experiential Learning 
<img src="https://eastgateacademy.ca/wp-content/uploads/2021/12/Learning.jpg" width="509" height="339">

There are many online resources that can help you learn different programming languages, such as Youtube, Stack Overflow and many more. By gaining the experience of a variety of different coding languages, you have more flexibility when applying to future careers. After this, it is best to start creating your own projects such as websites or games with the knowledge you have gained from the resources. Problem solving is also very important for software developers. Therefore, writing code to solve problems and problem solving in general is a very good way to build experience.
